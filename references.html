References

Other Places to Look

This page gives references for other useful tools you might need in order to help create a website.

  • Code.org

    • Reference from the video

  • HTMLDog.com

    • Tutorials on HTML and CSS in levels like beginner, intermediate, and advanced. Highly recommended for first time users

  • Codeceademy.com

    • Be apart of a community that teaches you about website coding, connects you with other people that you can ask questions to or are having similar problems, and gives you challenges to test your skills

  • W3Schools.com

    • More tutorials on CSS and HTML as well as many other website related topics. Also, has more specific links for things like headings, paragraphs, colors, etc.

  • JSFiddle.net

    • Code in one screen, see your results in another. (Very useful for playing around!)
